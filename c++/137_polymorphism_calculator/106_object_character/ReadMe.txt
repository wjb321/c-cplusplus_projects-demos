1.多态的好处就是组织结构清晰，就是每个不同功能能够用不同类进行区分
而且是模块化的，当进行调试的时候是整块进行调试，不需要对源码进行修改

2. 可读性很强

3. 前期和后期的扩展和维护自由度很高
